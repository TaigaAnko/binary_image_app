# はじめに
このソフトウェアは、PNG、JPG、JPEGの画像を二値化し、標準偏差のスコアを計算するアプリケーションです。

# 使い方
1. アプリケーションを開きます。
2. メニューから「ファイル」→「開く」を選択します。
3. PNG、JPG、JPEGの画像ファイルを選択します。
4. アプリケーションは選択された画像を即座に二値化し、標準偏差のスコアを計算します。
5. 標準偏差が高いほど、画像のコントラストが高いことを示します。
